## üöÄ Quick Start

Each module is self-contained and can be run independently.

### Web Development

```bash
cd mobileModuleXX
pnpm install
pnpm web
```

### Docker

```bash
cd mobileModuleXX
docker compose up
```

The application will be available at `http://localhost:8081`

> ‚ö†Ô∏è **Note for Windows users**: Docker performance on Windows is extremely slow due to file system synchronization between Windows and the Linux container. For better performance, consider using the web development method instead.
